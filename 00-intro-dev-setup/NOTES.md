<h2>Installing the dev environment</h2>
```
// Install
$ npm i -g typescript

// To run
$ tsc -v

// To compile
$ tsc index.ts

// Configure compiler
// See tsconfig.json
$ tsc --init

// Can now run all files with -w to watch for changes in .ts files
$ tsc -w

// Debugging
// See breakpoints in index.ts
// Run and Debug

// VS Code Extensions
